ආරම්භය මූලික ගැටලු විසදීම පාඨ ප්‍රදර්ශනය විචල්‍ය Substring ලයිස්තු නාමාවලි IF ප්‍රකාශනය WHILE ප්‍රකාශනය පුනර්කරණ පාලනය FOR ප්‍රකාශනය FUNCTION භාවිතය

ලැයිස්තු ( Lists )

විවිධ දත්ත ප්‍රරුප වලට අයත් දත්ත අයිතම එකතුවක් ගබඩා කිරීම සදහා ලැයිස්තු භාවිතා කරයි.

මුලින්ම List එකක් සාදන ආකාරය බලමු.

>>> s=['kamal',5,(4+4j)]

මෙහිදී List එක නිර්මානය වේ.


පහත ආකාරයෙන් List එක සෑදී ඇත් දැයි බලමු.

>>> s

['kamal', 5, (4+4j)]


List එ‍කේ ඇති අගයක් ලබා ගැනීමට

>>> s[1]

5


List එ‍කේ ඇති අගයන්ගෙන් අගයන් පරාසයක් වෙන් කර ගන්නා ආකාරය බලමු.

>>> s[0:2]

['kamal', 5]


මුලින්ම List එකේ අගයක් වෙනස් කරන ආකාරය බලමු.

>>> s[1]='sunil'

මෙහිදී List එ‍කේ 2 වන අගය ලෙස ඇති 5 යන දත්තය 'sunil' ලෙස වෙනස් වේ.


>>> s

['kamal', 'sunil', (4+4j)]


List එ‍කට අලුතෙන් දත්තයක් එකත කරන ආකාරය බලමු.

>>> s.append(16)

මෙහිදී අලුතෙන් 16 අයිතමය List එකට ඇතලත් වේ.
මෙහිදී අලුතෙන් දත්තයක් List එකට ඇතලත් කිරීමට append() භාවිතා කරයි.
List එ‍කේ නම.append(අලුතෙන් ඇතලු කරන අගය)
['kamal', 5, (4+4j), 16]


>>> s

['kamal', 'sunil', (4+4j), 16]



List කිහිපයක් එකතු කරන ආකාරය බලමු.


s නම් List එ‍කට අලුතෙන් තවත් List එ‍කක් සම්භන්ධ කරන ආකාරය බලමු.

>>> s=s+[1,'saman']

මෙහිදී s නම් List එ‍කත් [1,'saman'] List එ‍කත් එකට එකතු වේ.
'+' ලකණ ඒ සදහා භාවිතා කරයි.


>>> s

['kamal', 'sunil', (4+4j), 16, 1, 'saman']

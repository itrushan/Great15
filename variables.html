විචල්‍ය ( variable )

විචල්‍යයක් යනු පරිගණක වැඩසටහනක අගයන් තැන්පත් කිරීම සඳහා භාවිතා කරන පරිගණක මතක ස්ථානවලට දෙන නමකි.

පයිතන් භාෂාවේ විචල්‍ය භාවිතයේ දී අවධානය යොමුකළ යුතු කරුණු කිහිපයකි.
  • පයිතන් භාෂාවේ විචල්‍ය හදුන්වාදීම ( Declare ) අවශ්‍ය නොවේ.
  • පයිතන් භාෂාවේ විචල්‍ය වලට අගයන් පැවරීම පමනක් කළ හැකිය.
පයිතන් භාෂාවේ විචල්‍ය සදහා පහත ආකාරයෙන් අගයන් පැවරිය හැක.
